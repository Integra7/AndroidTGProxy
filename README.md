# WSProxy

WSProxy - Android приложение, которое помогает включить локальный прокси и быстро добавить его в Telegram через ссылку.

## How to use

- Установи приложение
- Открой WSProxy
- Включи прокси кнопкой Start
- Нажми Open Telegram и подтверди добавление прокси в Telegram
- Если нужно, включи Autostart after reboot

## Сборка

- Debug APK: `gradle :app:assembleDebug`
- Release AAB (Google Play): `gradle :app:bundleRelease`
- Release APK (для загрузки в магазины, где нужен APK): `gradle :app:assembleRelease`
